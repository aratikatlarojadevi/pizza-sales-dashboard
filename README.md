# pizza-sales-dashboard
Pizza Sales Report Dashboard

PROJECT TITLE:
Pizza Sales Analysis Dashboard – Power BI

SHORT DESCRIPTION:
This Power BI project analyzes pizza sales data to provide business insights into revenue, order trends, best-selling pizzas, customer ordering patterns, and sales performance by category and size. The dashboard helps stakeholders identify peak sales periods, top-performing products, and areas needing improvement through interactive visualizations and KPIs.

FEATURES:
Interactive Power BI dashboard with category filtering
Revenue, orders, and sales KPI tracking
Daily and monthly sales trend analysis
Best-selling and worst-selling pizza analysis
Sales breakdown by pizza category and pizza size
Visual comparison of quantity, revenue, and total orders

KEY INSIGHTS:
Highest number of orders are placed on weekends, especially Friday and Saturday evenings.
July to January recorded the highest monthly sales activity.
Classic category contributes the highest sales and orders.
Large-sized pizzas generate the maximum sales revenue.
Thai Chicken Pizza and Classic Deluxe Pizza are among the top-performing pizzas.

OBJECTIVE:
The main objective of this project is to help pizza businesses make data-driven decisions by analyzing sales performance, customer behavior, and product demand patterns using Power BI dashboards.

CONCLUSION:
This dashboard provides a complete overview of pizza sales performance and helps identify business opportunities through interactive analytics and visual storytelling.
